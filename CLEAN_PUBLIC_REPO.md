# Clean public SN126 Poker44 model repo

This repository is a clean-history public release of EmmanuelGreene SN126 Poker44 model miner implementation for manifest verification.

Production manifest should point to this repository and an exact commit hash. Evaluators can verify the implementation files and model artifact directly from GitHub/raw URLs at that commit.

Runtime-only deployment files such as PM2 ecosystem config, wallet names, hotkeys, and shell startup scripts are intentionally excluded from git.
